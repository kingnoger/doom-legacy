Doom Legacy bugs, unfinished stuff and excuses

  • Textures in the game are rendered incorrectly (software) or not at all (hardware). Caused by the char->bool conversion in the video unit?
  • Network code is only partially functional right now.
  • FS is not up-to-date with the rest of the code. It should be updated before re-enabling it.
  • Serialization unit is unfinished, saving and loading is disabled.
  • MAPINFO parser is disabled for now.
  • Skins are disabled until an "Actor representation class" is designed.
  • Heretic inventory use has some bugs.
  • Sometimes crashes when respawning after being killed by a monster.
