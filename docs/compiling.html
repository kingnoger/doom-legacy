How to compile Doom Legacy C++

Compiling the SDL version

You need to do the following stuff:
  • Get GCC 3.2, the Gnu Compiler Collection. Linux systems most likely have it installed. For Windows32, download and install MinGW 2.0 and MSYS.
  • Download and install SDL 1.2.5 development libraries, which include the runtime library (dll).

    The Windows binary version can be installed by decompressing the file i386-mingw32msvc.tar.gz to a temporary directory and then copying the 'include' and 'lib' directories to your MinGW directory. You also need to copy the file 'SDL.dll' to your windows/SYSTEM directory.

  • Download and install SDL_mixer-1.2.3 development libraries, using the same procedure as above.

    Windows32 users, note that 'SDL_mixer.h' needs to go to the 'MinGW/include/SDL' directory, not just 'MinGW/include'!

  • Download the doomlegacy_c++ branch from the CVS.
  • Win32: Start the MSYS shell. Go to the doomlegacy_c++ directory and say 'make' to build the debugging version or 'make OPT=1' to build the optimized version.
