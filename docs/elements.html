Features

  • Game engine has been rewritten using classes:
    • Can run several maps simultaneously
    • New game objects with custom AI are easy to make
    • Any creature can be played, not just Marine
    • Level ordering is free, one level can in principle have any number of exits that lead to different levels.
    • Damage system with different types of damage
    • Weapon groups á la Half-Life
    • ** general graphic presentations, MD3 support
    • ** DLL modifiability
  • HUD widgets
  • Other features
    • * Hexen/ZDoom-style MAPINFO support
    • * MapInfo parser rewritten (can read almost anything)
    • WAD2/WAD3 support
  • SW sound renderer rewritten
  • SDL as the multimedia interface:
    • Same code works for both Win32 and Linux
    • Can play back MUS, MIDI, Ogg Vorbis, probably also MP3 files
* Implemented but doesn't affect the game yet.
** Partly implemented
